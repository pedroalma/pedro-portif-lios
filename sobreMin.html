Sobre mim

Estou no começo da minha jornada no desenvolvimento web e atualmente estou aprendendo HTML e CSS no curso de Análise e Desenvolvimento de Sistemas do SENAC Largo Treze. O conteúdo tem sido bastante interessante, especialmente porque consigo ver como a estrutura de uma página web começa a tomar forma com apenas algumas linhas de código. No HTML, estou aprendendo a criar a estrutura básica de uma página, como usar as tags para títulos, parágrafos, listas e links. Já o CSS está me mostrando como estilizar essas páginas, aplicando cores, fontes e até mesmo criando layouts responsivos, que se ajustam dependendo do tamanho da tela. Embora eu ainda esteja no início, consigo perceber como essas duas linguagens são fundamentais para qualquer site. Estou empolgado para aprender mais, experimentar novos estilos e, quem sabe, construir algo meu no futuro!
